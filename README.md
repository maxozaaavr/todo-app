# ToDo API in Go
Features:
- Create list
- Get all ists
- Get list by ID
- Update list
- Delete list

- Create item of list
- Get all itmes
- Get item by ID
- Update item
- Delete item

# Technologies
- Programming language: Go
- Web framework: Gin
- Database: PostgreSQL
- Database migration
- RESTful JSON API

# Start the server with:
- `go run cmd/main.go`
